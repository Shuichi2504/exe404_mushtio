# Mushtio Web

Static web dashboard that talks directly to the IoTAgriculture backend API.

## Run

Open `index.html` with a static server from this folder:

```powershell
python -m http.server 5173
```

Then browse to `http://localhost:5173`.

The default API is the deployed backend:

```text
https://mushtio1-a2hcerawcpaqezb3.centralindia-01.azurewebsites.net/api
```

You can change it in the login screen before signing in.
